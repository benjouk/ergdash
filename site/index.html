What's in it

  • Dashboard — season metres, weekly volume, pace trend, personal bests, calendar heatmap, weekly time-in-zone, fitness sparkline.
  • Session detail — pace ribbon heatmap, stroke-level charts, interval reps with HR recovery, rate-vs-pace scatter, computed metrics (fade index, consistency, effort, DPS, watts/beat, HR drift).
  • Workouts — filterable, sortable table with CSV export.
  • Progress — fitness modelling, power-duration curve with ghost overlay, polarization stacks, efficiency trends, cumulative metres race line, fade fingerprint.
  • Yours — runs in Docker on your own hardware, backed by SQLite. No third party sees your training data.

Self-hosting

ErgDash runs as a single Docker container backed by SQLite. It talks to the Concept2 Logbook API via OAuth2 — a few minutes of one-time setup, documented in the README.

cp .env.example .env
# fill in C2_CLIENT_ID / C2_CLIENT_SECRET
docker compose up -d
